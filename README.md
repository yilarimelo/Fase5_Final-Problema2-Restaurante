# Proyecto: Gestión de Precios del Menú - Problema 2

Este proyecto soluciona la gestión de promociones para un restaurante utilizando programación en Python. 

## Descripción del Problema
El objetivo es aplicar un descuento del 15% a los productos de la categoría "Comida" cuyo precio base sea superior a un umbral de 18,000.

## Características del Código
- **Uso de Matriz:** Almacenamiento de datos estructurados (nombre, categoría, precio base).
- **Modularidad:** Función `calcular_precio_final` para aplicar la lógica del descuento.
- **Estructuras de Control:** Uso de ciclos `for` y condicionales `if/else`.

## Cómo ejecutarlo
1. Asegúrate de tener Python instalado.
2. Ejecuta el archivo `Fase 5.py` en tu terminal.
